Превращение букв в цифры

текстовые символы (буквы, цифры, знаки препинания) преобразуются в числа с помощью таблиц кодировки символов

Основные таблицы стандартов кодирования символов

ASCII
  • 1 символ - 7 бит
  • Алфавит - 128 символов
Extended ASCII
  • 1 символ - 8 бит
  • Алфавит - 256 символов
Unicode (UTF-8/16/32)
  • 1 символ - 8-32 бита
  • Алфавит - 1,112,064 символов

dict1 = {'A' : 1, 'B' : 2, 'C' : 3, 'D' : 4, 'E' : 5,
        'F' : 6, 'G' : 7, 'H' : 8, 'I' : 9, 'J' : 10,
        'K' : 11, 'L' : 12, 'M' : 13, 'N' : 14, 'O' : 15,
        'P' : 16, 'Q' : 17, 'R' : 18, 'S' : 19, 'T' : 20,
        'U' : 21, 'V' : 22, 'W' : 23, 'X' : 24, 'Y' : 25, 'Z' : 26}
 
dict2 = {0 : 'Z', 1 : 'A', 2 : 'B', 3 : 'C', 4 : 'D', 5 : 'E',
        6 : 'F', 7 : 'G', 8 : 'H', 9 : 'I', 10 : 'J',
        11 : 'K', 12 : 'L', 13 : 'M', 14 : 'N', 15 : 'O',
        16 : 'P', 17 : 'Q', 18 : 'R', 19 : 'S', 20 : 'T',
        21 : 'U', 22 : 'V', 23 : 'W', 24 : 'X', 25 : 'Y'}
 
def encrypt(message, shift):
    cipher = ''
    for letter in message:
        if(letter != ' '):
            num = ( dict1[letter] + shift ) % 26
            cipher += dict2[num]
        else:
            # adds space
            cipher += ' '
 
    return cipher
 
def decrypt(message, shift):
    decipher = ''
    for letter in message:
        if(letter != ' '):
            num = ( dict1[letter] - shift + 26) % 26
            decipher += dict2[num]
        else:
            decipher += ' '
 
    return decipher

Имплементация


ALPHABET_LENGTH=26
def generate_key(msg, key):
    key = list(key)
    if len(msg) == len(key):
        return key
    else:
        for i in range(len(msg) - len(key)):
            key.append(key[i % len(key)])
    return "".join(key)

def encrypt_vigenere(msg, key):
    encrypted_text = []
    key = generate_key(msg, key)
    for i in range(len(msg)):
        char = msg[i]
        if char.islower():
            encrypted_char = chr((ord(char) + ord(key[i]) % ALPHABET_LENGTH))
# Номер буквы сообщения + номер буквы ключа % длина алфавита
        else:
            encrypted_char = char
        encrypted_text.append(encrypted_char)
    return "".join(encrypted_text)

def decrypt_vigenere(msg, key):
    decrypted_text = []
    key = generate_key(msg, key)
    for i in range(len(msg)):
        char = msg[i]
        if char.islower():
            decrypted_char = chr((ord(char) - ord(key[i]) - 1))
        else:
            decrypted_char = char
        decrypted_text.append(decrypted_char)
    return "".join(decrypted_text)

DES

Data Encryption Standard

симметричный алгоритм шифрования, разработанный в 1970-х годах компанией IBM и утверждённый Национальным институтом стандартов и технологий США (NIST) в 1977 году

Характеристика

Длина ключа

использует ключ длиной 56 бит

Блочный алгоритм

работает с блоками данных длиной 64 бита

Симметричность

Один и тот же ключ используется как для шифрования, так и для расшифровки

Сложность

DES выполняет 16 раундов преобразований, включая перестановки и замену данных
